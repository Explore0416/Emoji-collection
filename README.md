# Emoji-collection
**Emoji-collection**

**项目简介**

Emoji-collection 是一个表情包素材收集与网页端展示项目，基于 HTML 与 JavaScript 开发，用于集中整理、浏览表情包素材，提供轻量化的表情资源展示与管理能力。
